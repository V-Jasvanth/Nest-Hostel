🏠 Nest Hostel Management System

Nest Hostel is a web-based hostel management system designed to simplify and digitize hostel operations. It provides a centralized platform where Admins, Students, Wardens, and Guardians can log in and access role-based functionalities efficiently.

🌐 Live Demo

👉 https://nest-hostel.web.app

✨ Features
🔐 Multi-role login system (Admin / Student / Warden / Guardian)
🧑‍🎓 Student profile and hostel information management
🧑‍🏫 Warden dashboard for monitoring activities
👨‍👩‍👧 Guardian access to track student details
🏢 Room allocation and management
⚡ Real-time data handling using Firebase
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Database: Firebase Realtime Database
Hosting: Firebase Hosting
