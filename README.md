Nest Hostel Management System
📌 Overview

Nest Hostel is a web-based hostel management system designed to simplify and digitize hostel operations. It provides a centralized platform where Admins, Students, Wardens, and Guardians can log in and access role-based functionalities. The system helps in managing student records, room allocation, and communication efficiently.

🚀 Features
🔐 Multi-role login system (Admin / Student / Warden / Guardian)
👨‍💼 Admin dashboard to manage students and rooms
🧑‍🎓 Student panel to view personal and hostel details
🧑‍🏫 Warden access for monitoring and management
👨‍👩‍👧 Guardian access to track student information
☁️ Real-time data handling using Firebase
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Database: Firebase Realtime Database
Tools: VS Code, GitHub
